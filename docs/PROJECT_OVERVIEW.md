# BizReels — Project Overview

BizReels is a production-ready, highly secure local social commerce platform tailored specifically for the Indian marketplace. The application connects local vendors, content creators, and customers directly, enabling local discovery, seamless communication, direct bargaining, requirement matchmaking, and credit-based interactions.

## Core Value Proposition

The platform's core values are:
* **Discover Local**: Locate verified physical shops, service providers, and local creators in the user's vicinity using geolocated search and category matching.
* **Chat Direct**: Facilitate real-time, direct messaging between customers and vendors without intermediate platform interference.
* **Deal Fair**: Allow customers and vendors to negotiate deals dynamically using an interactive offer-negotiation flow, culminating in mutually agreed-upon terms.

## User Roles & Capabilities

BizReels operates on a multi-role architecture:

| Role | Primary Activities | Access Rights |
| :--- | :--- | :--- |
| **Customer** | Browse local listings, view video reels, chat with vendors, negotiate deals, submit buying requirements, review products/services. | Access to customer facing screens, wallet operations, review submissions, chat threads. |
| **Vendor** | Create and manage product/service listings, receive customer requirement notifications, submit custom proposals, negotiate deals, view shop analytics. | Access to Vendor Dashboard, active listings management, analytics metrics, proposals. |
| **Creator** | Upload local content reels, tag local listings, promote neighborhood vendors, earn rewards. | Access to Creator Dashboard, reel upload, content tagging modules. |
| **Admin** | Moderates listings, reviews submitted KYC documents, bans/unbans users, resolves disputes/reports, manages system-wide integrations. | Access to Admin Console, user management, KYC approvals, settings panel. |

## Core Platform Modules

1. **Local Social Feed & Reels**: A TikTok/Instagram-like localized video and image feed where creators showcase products and local shops, helping buyers discover what is nearby.
2. **Dynamic Listings Catalog**: Vendors publish structured listings categorized by type (New Product, Old Product, or Service) with location coordinates, tags, and variants.
3. **Requirement & Proposal Matchmaker**: Customers post buying requests (e.g., "Need a wedding photographer"). AI parses the demand and matches nearby vendors who can submit custom proposals.
4. **Real-time Deals Negotiation**: Interactive chat workspace supporting immediate text messaging, location sharing, listing cards, and system-driven offer/counter-offer negotiation deals.
5. **Trust+ & KYC**: System verifies seller/vendor identity through official documents (Aadhaar, PAN, etc.) and issues "Trust Badges", elevating their trust score.
6. **Unified Wallet & Payments**: Real-time credits and INR wallet. Credits are spent for premium features (e.g. phone reveals, listing boosts), refilled using Razorpay payments.
